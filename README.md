# pizza-sales-project
SQL Server and Excel dashboard analyzing one year of pizza sales data.

## Overview

This project analyzes 48,600 pizza sales records using SQL Server and Microsoft Excel.

The objective was to calculate key performance indicators (KPIs), analyze sales trends, and build an interactive dashboard for business reporting.

## Tools

- SQL Server Management Studio (SSMS)
- SQL
- Microsoft Excel
- PivotTables
- PivotCharts
- Timeline Slicers

## Skills

- Data Cleaning
- Data Processing
- Data Analysis
- SQL Querying
- Dashboard Design
